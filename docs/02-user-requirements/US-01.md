# US-01: Регистрация на платформе

**Идентификатор:** US-01

**Формулировка:**
Я, как Пользователь, хочу зарегистрироваться на платформе, указав email и пароль (или через соцсети), чтобы получить доступ к функциям системы в соответствии с моей ролью.

**Роли:**
- Неаутентифицированный Пользователь

**Атрибуты:**
- Источник: lab5, User Story 1
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [FR-AU-01](../03-functional-requirements/FR-AU-01.md) — Валидация данных при регистрации
- [FR-AU-04](../03-functional-requirements/FR-AU-04.md) — Регистрация и вход через социальные сети
- [NFR-SEC-01](../04-non-functional-requirements/NFR-SEC-01.md) — Безопасное хранение паролей
