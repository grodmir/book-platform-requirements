# US-06: Взаимодействие с читателями через комментарии

**Идентификатор:** US-06

**Формулировка:**
Я, как Автор, хочу общаться с читателями через раздел «Комментарии» под каждой книгой и отвечать на их вопросы, чтобы выстраивать доверительные отношения с аудиторией.

**Роли:**
- Автор
- Читатель (для создания комментария)

- Приоритет: High
- Стабильность: High
