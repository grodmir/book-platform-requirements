# US-10a: Инициирование покупки книги

**Идентификатор:** US-10a

**Формулировка:**
Я, как Читатель, хочу нажать кнопку «Купить» на карточке книги, чтобы система перенаправила меня на защищённую страницу платёжного шлюза.

**Роли:**
- Читатель

**Атрибуты:**
- Источник: lab5, User Story 10 (декомпозировано)
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [FR-PAY-01a](../03-functional-requirements/FR-PAY-01a.md) — Перенаправление на платёжный шлюз
- [FR-PAY-02](../03-functional-requirements/FR-PAY-02.md) — Проверка наличия доступа перед покупкой
- [NFR-SEC-03](../04-non-functional-requirements/NFR-SEC-03.md) — Безопасная передача платёжных данных
