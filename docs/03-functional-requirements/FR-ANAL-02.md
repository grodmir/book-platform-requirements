# FR-ANAL-02: Детальная статистика по книге

**Идентификатор:** FR-ANAL-02

**Формулировка:**
Система должна при выборе автором конкретной книги отображать график динамики продаж и просмотров (по дням/неделям) и воронку чтения (сколько читателей дочитало до 10%, 50%, 90%) с возможностью фильтрации по периоду.

**Атрибуты:**
- Источник: lab6, UC-04
- Приоритет: High
- Стабильность: High
