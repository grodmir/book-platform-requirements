# FR-ANAL-03: Отображение читателей онлайн

**Идентификатор:** FR-ANAL-03

**Формулировка:**
Система должна отображать количество читателей, находящихся на страницах книг автора в данный момент, с обновлением в реальном времени. При недоступности сервиса реального времени виджет заменяется информационным сообщением.

- Приоритет: High
- Стабильность: Medium
