# FR-ANAL-03: Отображение читателей онлайн

**Идентификатор:** FR-ANAL-03

**Формулировка:**
Система должна отображать количество читателей, находящихся на страницах книг автора в данный момент, с обновлением в реальном времени. «Онлайн» определяется как активность на странице книги за последние 30 секунд. При недоступности сервиса реального времени виджет заменяется информационным сообщением.

**Атрибуты:**
- Источник: lab6, UC-04; lab9, Раздел 4.2
- Приоритет: High
- Стабильность: Medium
