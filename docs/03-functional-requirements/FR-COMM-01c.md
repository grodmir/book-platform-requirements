# FR-COMM-01: Создание и отображение комментариев

**Идентификатор:** FR-COMM-01

**Формулировка:**
Система должна предоставлять читателю возможность оставлять комментарий под книгой, а автору — отвечать на них. Комментарии и ответы отображаются в виде иерархической структуры с пометкой «Автор» для ответов автора.

**Атрибуты:**
- Источник: lab5, User Story 6; lab6, UC-05
- Приоритет: High
- Стабильность: High

**Связанные требования:**
- [US-06](../02-user-requirements/US-06.md) — Взаимодействие с читателями через комментарии
- [FR-COMM-02](../03-functional-requirements/FR-COMM-02.md) — Модерация комментариев
