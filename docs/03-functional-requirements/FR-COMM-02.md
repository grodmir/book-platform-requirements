# FR-COMM-02: Модерация комментариев

**Идентификатор:** FR-COMM-02

**Формулировка:**
Система должна автоматически фильтровать комментарии на наличие нецензурной лексики. При обнаружении стоп-слов система отклоняет публикацию комментария с предупреждением.

**Атрибуты:**
- Источник: lab6, UC-10
- Приоритет: High
- Стабильность: Medium
