# FR-COMM-02: Модерация комментариев

**Идентификатор:** FR-COMM-02

**Формулировка:**
Система должна автоматически фильтровать комментарии на наличие нецензурной лексики. Список запрещённых слов содержит 100 наиболее распространённых нецензурных слов русского языка, хранится в конфигурационном файле и обновляется при каждом релизе. При обнаружении любого слова из списка система отклоняет публикацию комментария с сообщением «Комментарий содержит запрещённые слова».

- Приоритет: High
- Стабильность: Medium
