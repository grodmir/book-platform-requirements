# FR-CONT-01: Валидация файла при загрузке

**Идентификатор:** FR-CONT-01

**Формулировка:**
Система должна проверять расширение и MIME-тип загружаемого файла (допустимы только PDF и EPUB) и его размер (не более 50 МБ). При несоответствии система отклоняет файл с сообщением об ошибке.

**Атрибуты:**
- Источник: lab6, UC-02
- Приоритет: Critical
- Стабильность: High
