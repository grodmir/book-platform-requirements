# FR-CONT-02: Проверка обязательных полей метаданных

**Идентификатор:** FR-CONT-02

**Формулировка:**
Система должна блокировать публикацию книги, если не заполнены обязательные поля метаданных (название, жанр). Незаполненные поля подсвечиваются, кнопка «Опубликовать» блокируется.
