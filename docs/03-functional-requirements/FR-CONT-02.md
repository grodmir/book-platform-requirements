# FR-CONT-02: Проверка обязательных полей метаданных

**Идентификатор:** FR-CONT-02

**Формулировка:**
Система должна блокировать публикацию книги, если не заполнены обязательные поля метаданных (название, жанр). Незаполненные поля подсвечиваются красным, кнопка «Опубликовать» неактивна.

**Атрибуты:**
- Источник: lab6, UC-02
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [US-03b](../02-user-requirements/US-03b.md) — Заполнение метаданных произведения
- [NFR-USA-02](../04-non-functional-requirements/NFR-USA-02.md) — Простота публикации для начинающих авторов
