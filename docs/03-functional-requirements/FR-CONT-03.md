# FR-CONT-03: Сохранение черновика

**Идентификатор:** FR-CONT-03

**Формулировка:**
Система должна позволять автору сохранить книгу как черновик на любом этапе мастера публикации. Черновик виден только автору со статусом «Черновик» и доступен для последующего редактирования и публикации. Максимальное количество черновиков на одного автора — 50. Черновики хранятся бессрочно.

- Приоритет: Medium
- Стабильность: Medium
