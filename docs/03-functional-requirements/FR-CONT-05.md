# FR-CONT-05: Публикация по главам

**Идентификатор:** FR-CONT-05

**Формулировка:**
Система должна предоставлять автору возможность публиковать произведение по главам. Первые главы могут быть отмечены как бесплатные, остальные — платные. Читатель оплачивает доступ один раз и получает все опубликованные и будущие главы.

- Стабильность: Medium
