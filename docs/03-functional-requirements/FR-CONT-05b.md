# FR-CONT-05b: Платная книга по главам

**Идентификатор:** FR-CONT-05b

**Формулировка:**
Система должна позволять автору установить цену на книгу, публикуемую по главам. Читатель платит один раз за книгу и получает доступ ко всем текущим и будущим главам. Доступ не требует отдельной оплаты за каждую главу.

**Атрибуты:**
- Источник: lab5, User Story 7; lab6, UC-02 (декомпозировано)
- Приоритет: Medium
- Стабильность: Medium

**Связанные требования:**
- [US-07b](../02-user-requirements/US-07b.md) — Публикация платных глав
- [FR-MON-01a](../03-functional-requirements/FR-MON-01a.md) — Установка цены
