# FR-FAV-01: Управление избранным

**Идентификатор:** FR-FAV-01

**Формулировка:**
Система должна предоставлять читателю возможность добавлять книги в раздел «Избранное» и удалять их оттуда. Список избранного отображается в личном кабинете в порядке добавления с возможностью перехода к карточке книги.

**Атрибуты:**
- Источник: lab5, User Story 13
- Приоритет: Medium
- Стабильность: High

**Связанные требования:**
- [US-13](../02-user-requirements/US-13.md) — Добавление книги в «Избранное»
