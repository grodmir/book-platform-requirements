# FR-FAV-01b: Удаление из избранного

**Идентификатор:** FR-FAV-01b

**Формулировка:**
Система должна предоставлять читателю возможность удалять книги из раздела «Избранное» одним нажатием кнопки «Удалить из избранного».

**Атрибуты:**
- Источник: lab5, User Story 13 (декомпозировано)
- Приоритет: Medium
- Стабильность: High

**Связанные требования:**
- [US-13](../02-user-requirements/US-13.md) — Добавление книги в «Избранное»
