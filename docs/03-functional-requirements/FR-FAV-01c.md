# FR-FAV-01c: Отображение избранного

**Идентификатор:** FR-FAV-01c

**Формулировка:**
Система должна отображать список избранных книг в личном кабинете читателя в порядке добавления (сначала новые) с отображением обложки, названия, автора и кнопки «Удалить».

**Атрибуты:**
- Источник: lab5, User Story 13 (декомпозировано)
- Приоритет: Medium
- Стабильность: High

**Связанные требования:**
- [US-13](../02-user-requirements/US-13.md) — Добавление книги в «Избранное»
