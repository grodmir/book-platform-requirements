# FR-LIB-01: Моя библиотека

**Идентификатор:** FR-LIB-01

**Формулировка:**
Система должна отображать в личном кабинете читателя раздел «Моя библиотека» со списком всех купленных книг. Для каждой книги отображается обложка, название, автор, прогресс чтения и кнопка «Читать», открывающая книгу с последней сохранённой позиции.

**Атрибуты:**
- Источник: lab5, User Story 14
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [US-14](../02-user-requirements/US-14.md) — Доступ к списку купленных книг
- [FR-READ-01](../03-functional-requirements/FR-READ-01.md) — Отображение книг во встроенной читалке
- [FR-READ-03](../03-functional-requirements/FR-READ-03.md) — Сохранение и синхронизация прогресса чтения
- [FR-PAY-01](../03-functional-requirements/FR-PAY-01.md) — Проведение платежа через внешний шлюз
