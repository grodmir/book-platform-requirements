# FR-LIB-01c: Кнопка «Читать»

**Идентификатор:** FR-LIB-01c

**Формулировка:**
Система должна отображать для каждой книги в «Моей библиотеке» кнопку «Читать». При нажатии система открывает книгу во встроенной читалке с последней сохранённой позиции.

**Атрибуты:**
- Источник: lab5, User Story 14 (декомпозировано)
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [US-14a](../02-user-requirements/US-14a.md) — Отображение списка купленных книг
- [FR-READ-01](../03-functional-requirements/FR-READ-01.md) — Отображение книг во встроенной читалке
- [FR-READ-03](../03-functional-requirements/FR-READ-03.md) — Сохранение и синхронизация прогресса чтения
