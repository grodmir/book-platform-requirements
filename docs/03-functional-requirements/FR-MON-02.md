# FR-MON-02: Применение скидки

**Идентификатор:** FR-MON-02

**Формулировка:**
Система должна позволять автору устанавливать временную скидку на книгу в процентах (от 1% до 90%) с указанием периода действия (дата начала и дата окончания). По истечении периода цена автоматически возвращается к исходной.

**Атрибуты:**
- Источник: lab2, Сегмент «Авторы»
- Приоритет: Medium
- Стабильность: Medium

**Связанные требования:**
- [US-04](../02-user-requirements/US-04.md) — Установка цены на книгу
- [FR-MON-01a](../03-functional-requirements/FR-MON-01a.md) — Установка цены
