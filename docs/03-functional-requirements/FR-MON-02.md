# FR-MON-02: Применение скидки

**Идентификатор:** FR-MON-02

**Формулировка:**
Система должна позволять автору устанавливать временную скидку на книгу с указанием периода действия. По истечении периода цена автоматически возвращается к исходной.

**Атрибуты:**
- Источник: lab2, Сегмент «Авторы»
- Приоритет: Medium
- Стабильность: Medium
