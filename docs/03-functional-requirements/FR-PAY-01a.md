# FR-PAY-01a: Перенаправление на платёжный шлюз

**Идентификатор:** FR-PAY-01a

**Формулировка:**
Система должна перенаправлять читателя на защищённую страницу внешнего платёжного сервиса YooKassa для ввода данных карты и проведения транзакции.

**Атрибуты:**
- Источник: lab6, UC-08 (декомпозировано)
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [US-10a](../02-user-requirements/US-10a.md) — Инициирование покупки книги
- [US-15](../02-user-requirements/US-15.md) — Бесплатное чтение фрагмента книги
- [NFR-SEC-03](../04-non-functional-requirements/NFR-SEC-03.md) — Безопасная передача платёжных данных
