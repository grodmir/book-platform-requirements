# FR-PAY-01b: Обработка успешного платежа

**Идентификатор:** FR-PAY-01b

**Формулировка:**
Система должна получать webhook-уведомление от YooKassa при успешной оплате, после чего открыть доступ к книге для читателя и добавить книгу в раздел «Моя библиотека».

**Атрибуты:**
- Источник: lab6, UC-08 (декомпозировано)
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [US-10b](../02-user-requirements/US-10b.md) — Получение доступа после оплаты
- [FR-LIB-01a](../03-functional-requirements/FR-LIB-01a.md) — Отображение списка купленных книг
