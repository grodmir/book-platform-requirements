# FR-PAY-02: Проверка наличия доступа перед покупкой

**Идентификатор:** FR-PAY-02

**Формулировка:**
Система должна перед проведением платежа проверять, нет ли у читателя уже активного доступа к книге. Если доступ есть, система скрывает кнопку «Купить» и отображает кнопку «Читать».

**Атрибуты:**
- Источник: lab6, UC-08
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [US-10a](../02-user-requirements/US-10a.md) — Инициирование покупки книги
