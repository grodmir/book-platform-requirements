# FR-PAY-04: Сохранение карты для покупки в один клик

**Идентификатор:** FR-PAY-04

**Формулировка:**
Система должна позволять читателю сохранить данные банковской карты в профиле и выполнять покупку в один клик с подтверждением в модальном окне без повторного ввода данных.

**Атрибуты:**
- Источник: lab6, UC-08
- Приоритет: Medium
- Стабильность: Medium

**Связанные требования:**
- [US-10](../02-user-requirements/US-10.md) — Покупка книги
- [FR-PAY-01](../03-functional-requirements/FR-PAY-01.md) — Проведение платежа через внешний шлюз
- [NFR-SEC-03](../04-non-functional-requirements/NFR-SEC-03.md) — Безопасная передача платёжных данных
