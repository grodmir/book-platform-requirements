# FR-PAY-04b: Покупка в один клик

**Идентификатор:** FR-PAY-04b

**Формулировка:**
Система должна позволять читателю, у которого сохранён токен карты, выполнить покупку в один клик: при нажатии кнопки «Купить в один клик» отображается модальное окно с подтверждением, после подтверждения платеж проводится без повторного ввода данных карты.

**Атрибуты:**
- Источник: lab6, UC-08 (декомпозировано)
- Приоритет: Medium
- Стабильность: Medium

**Связанные требования:**
- [US-10a](../02-user-requirements/US-10a.md) — Инициирование покупки книги
- [FR-PAY-01a](../03-functional-requirements/FR-PAY-01a.md) — Перенаправление на платёжный шлюз
