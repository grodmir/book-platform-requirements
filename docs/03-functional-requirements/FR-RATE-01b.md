# FR-RATE-01b: Написание отзыва

**Идентификатор:** FR-RATE-01b

**Формулировка:**
Система должна позволять читателю написать текстовый отзыв к книге. Отзыв является опциональным и может быть оставлен как вместе с оценкой, так и после её выставления. Отзыв проходит модерацию через FR-COMM-02.

**Атрибуты:**
- Источник: lab6, UC-10 (декомпозировано)
- Приоритет: High
- Стабильность: High

**Связанные требования:**
- [US-12b](../02-user-requirements/US-12b.md) — Написание текстового отзыва
- [FR-COMM-02](../03-functional-requirements/FR-COMM-02.md) — Модерация комментариев
