# FR-RATE-02: Блокировка повторной оценки

**Идентификатор:** FR-RATE-02

**Формулировка:**
Система должна запрещать читателю оставлять более одной оценки для одной книги. При наличии оценки форма скрывается, отображается ранее выставленная оценка.

**Атрибуты:**
- Источник: lab6, UC-10
- Приоритет: High
- Стабильность: High

**Связанные требования:**
- [US-12](../02-user-requirements/US-12.md) — Оценка и отзыв на книгу
- [FR-RATE-01](../03-functional-requirements/FR-RATE-01.md) — Выставление оценки и отзыва
