# FR-RATE-02: Блокировка повторной оценки

**Идентификатор:** FR-RATE-02

**Формулировка:**
Система должна запрещать читателю оставлять более одной оценки для одной книги. При наличии оценки форма для выставления оценки скрывается, отображается ранее выставленная оценка с пометкой «Ваша оценка».

**Атрибуты:**
- Источник: lab6, UC-10
- Приоритет: High
- Стабильность: High

**Связанные требования:**
- [US-12a](../02-user-requirements/US-12a.md) — Выставление оценки книге
- [FR-RATE-01a](../03-functional-requirements/FR-RATE-01a.md) — Выставление оценки
