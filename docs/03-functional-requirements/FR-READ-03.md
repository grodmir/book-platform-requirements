# FR-READ-03: Сохранение и синхронизация прогресса чтения

**Идентификатор:** FR-READ-03

**Формулировка:**
Система должна автоматически сохранять позицию чтения при каждом перелистывании страницы. При открытии книги читатель продолжает с последней сохранённой позиции. Синхронизация прогресса между устройствами одного пользователя должна происходить в течение 5 секунд после сохранения позиции.

**Атрибуты:**
- Источник: lab5, User Story 11; lab2, Сегмент «Читатели»
- Приоритет: High
- Стабильность: High
