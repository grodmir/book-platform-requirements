# FR-READ-03: Сохранение и синхронизация прогресса чтения

**Идентификатор:** FR-READ-03

**Формулировка:**
Система должна автоматически сохранять позицию чтения и синхронизировать её между всеми устройствами пользователя. При открытии книги читатель продолжает с последней сохранённой позиции.
