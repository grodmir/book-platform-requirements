# FR-READ-04: Бесплатное чтение фрагмента

**Идентификатор:** FR-READ-04

**Формулировка:**
Система должна предоставлять доступ к чтению первых 10–15% любой книги без покупки. При достижении границы фрагмента отображается баннер с ценой и кнопкой «Купить полную версию». При ошибке загрузки книги система показывает сообщение и кнопку «Скачать файл».

**Атрибуты:**
- Источник: lab5, User Story 15; lab6, UC-08
- Приоритет: High
- Стабильность: High

- [FR-READ-01](../03-functional-requirements/FR-READ-01.md) — Отображение книг во встроенной читалке
- [FR-PAY-01](../03-functional-requirements/FR-PAY-01.md) — Проведение платежа через внешний шлюз
