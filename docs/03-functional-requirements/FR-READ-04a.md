# FR-READ-04a: Доступ к 10% книги

**Идентификатор:** FR-READ-04a

**Формулировка:**
Система должна предоставлять доступ к чтению первых 10% объёма любой книги без покупки для любого аутентифицированного читателя. Отсчёт процентов ведётся по количеству страниц или байт (в зависимости от формата). При достижении 10% чтение блокируется.

**Атрибуты:**
- Источник: lab5, User Story 15; lab6, UC-08 (декомпозировано)
- Приоритет: High
- Стабильность: High

**Связанные требования:**
- [US-15](../02-user-requirements/US-15.md) — Бесплатное чтение фрагмента книги
- [FR-READ-01](../03-functional-requirements/FR-READ-01.md) — Отображение книг во встроенной читалке
