# FR-READ-04b: Баннер покупки

**Идентификатор:** FR-READ-04b

**Формулировка:**
Система должна при достижении читателем границы бесплатного фрагмента (10% книги) отображать внутри читалки баннер с ценой полной версии и кнопкой «Купить полную версию». При нажатии на кнопку система инициирует процесс покупки.

**Атрибуты:**
- Источник: lab5, User Story 15; lab6, UC-08 (декомпозировано)
- Приоритет: High
- Стабильность: High

**Связанные требования:**
- [US-15](../02-user-requirements/US-15.md) — Бесплатное чтение фрагмента книги
- [FR-PAY-01a](../03-functional-requirements/FR-PAY-01a.md) — Перенаправление на платёжный шлюз
