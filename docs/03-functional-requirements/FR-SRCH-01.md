# FR-SRCH-01: Полнотекстовый поиск

**Идентификатор:** FR-SRCH-01

**Формулировка:**
Система должна предоставлять строку поиска, выполняющую полнотекстовый поиск по названиям книг и именам авторов. Результаты обновляются через 300 мс после остановки ввода пользователем (debounce). Результаты отображаются в виде списка карточек книг.

**Атрибуты:**
- Источник: lab5, User Story 9
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [US-09a](../02-user-requirements/US-09a.md) — Полнотекстовый поиск по названию и автору
- [NFR-PERF-02](../04-non-functional-requirements/NFR-PERF-02.md) — Время отклика страниц каталога и поиска
