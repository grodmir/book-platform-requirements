# FR-SRCH-02: Фильтрация по жанрам

**Идентификатор:** FR-SRCH-02

**Формулировка:**
Система должна предоставлять фильтр по жанрам (один или несколько). При выборе жанров каталог динамически обновляется, показывая книги, соответствующие хотя бы одному выбранному жанру.

- Источник: lab5, User Story 9
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [US-09b](../02-user-requirements/US-09b.md) — Фильтрация по жанрам
