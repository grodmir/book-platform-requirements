# FR-SRCH-03: Сортировка каталога

**Идентификатор:** FR-SRCH-03

**Формулировка:**
Система должна предоставлять сортировку каталога по следующим критериям: рейтинг (по убыванию), популярность (по количеству продаж за последние 30 дней), новизна (по дате публикации, сначала новые).

**Атрибуты:**
- Источник: lab5, User Story 9
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [US-09c](../02-user-requirements/US-09c.md) — Сортировка каталога
