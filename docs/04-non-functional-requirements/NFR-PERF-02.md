# NFR-PERF-02: Время отклика страниц каталога и поиска

**Идентификатор:** NFR-PERF-02

**Формулировка:**
Время отклика системы при поиске и фильтрации каталога книг не должно превышать 2 секунд для 95% запросов. Результаты поиска должны обновляться с задержкой не более 500 мс после ввода (с учётом debounce 300 мс из FR-SRCH-01).

**Атрибуты:**
- Источник: lab2, Сегмент «Читатели» (Раздражающие факторы)
- Приоритет: High
- Стабильность: Medium

**Связанные требования:**
- [US-09a](../02-user-requirements/US-09a.md) — Полнотекстовый поиск по названию и автору
- [FR-SRCH-01](../03-functional-requirements/FR-SRCH-01.md) — Полнотекстовый поиск
