# NFR-SUP-01b: Поддержка форматов

**Идентификатор:** NFR-SUP-01b

**Формулировка:**
Читалка должна корректно отображать книги в форматах PDF (версия 1.7) и EPUB (версия 3). Система должна обеспечивать возможность скачивания купленного файла на устройство пользователя в исходном формате без изменения содержимого.

**Атрибуты:**
- Источник: lab9, Раздел 1.7 (Зависимости); lab2, Сегмент «Читатели» (декомпозировано)
- Приоритет: Critical
- Стабильность: High

**Связанные требования:**
- [US-03a](../02-user-requirements/US-03a.md) — Загрузка файла произведения
- [US-11a](../02-user-requirements/US-11a.md) — Чтение книги в читалке
- [FR-CONT-01](../03-functional-requirements/FR-CONT-01.md) — Валидация файла при загрузке
- [FR-READ-01](../03-functional-requirements/FR-READ-01.md) — Отображение книг во встроенной читалке
