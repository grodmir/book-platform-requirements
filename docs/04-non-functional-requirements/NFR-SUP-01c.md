# NFR-SUP-01c: Скачивание файла

**Идентификатор:** NFR-SUP-01c

**Формулировка:**
Система должна позволять читателю скачать купленный файл книги на устройство в исходном формате (PDF или EPUB) без DRM-ограничений. Кнопка «Скачать» должна быть доступна на странице книги в «Моей библиотеке».

**Атрибуты:**
- Источник: lab2, Сегмент «Читатели» (декомпозировано)
- Приоритет: Medium
- Стабильность: High

**Связанные требования:**
- [US-11a](../02-user-requirements/US-11a.md) — Чтение книги в читалке
