# NFR-USA-02: Простота публикации для начинающих авторов

**Идентификатор:** NFR-USA-02

**Формулировка:**
Процесс публикации книги должен быть реализован в виде пошагового мастера с количеством шагов не более трёх. Каждый шаг должен содержать поясняющие подсказки. Время выполнения процесса публикации тестовым пользователем, не знакомым с системой, не должно превышать 3 минут без обращения к документации.

**Атрибуты:**
- Источник: lab2, Сегмент «Авторы»; lab9, Раздел 4.2
- Приоритет: High
- Стабильность: High
