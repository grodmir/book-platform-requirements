# NFR-USA-02: Простота публикации для начинающих авторов

**Идентификатор:** NFR-USA-02

**Формулировка:**
Процесс публикации книги должен быть реализован в виде пошагового мастера с количеством шагов не более трёх. Каждый шаг должен содержать поясняющие подсказки. Пользователь должен иметь возможность опубликовать книгу, не обращаясь к документации.

**Атрибуты:**
- Источник: lab2, Сегмент «Авторы»; lab9, Раздел 4.2
- Приоритет: High
- Стабильность: High

- [FR-CONT-03](../03-functional-requirements/FR-CONT-03.md) — Сохранение черновика
